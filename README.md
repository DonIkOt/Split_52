# 💰 СплитЧек

> Делим счёт честно — считает кто кому сколько должен.

---

## Возможности

- 🧾 **Несколько чеков** в одной сессии (магазин + кафе)
- 👥 **Гибкие доли** — по умолчанию поровну, корректируются вручную
- 📷 **Сканирование чеков** через камеру (OCR, только онлайн-версия)
- 💸 **Минимум переводов** — алгоритм оптимизирует количество транзакций
- 📋 **Копировать итоги** — готовый текст для мессенджера
- 📊 **Экспорт** в JPG и Excel
- 💾 **История** — до 5 сессий, автоудаление через 30 дней
- 📵 **Офлайн** — после первого открытия работает без интернета

## Использование

### Онлайн (рекомендуется)

**🔗 https://YOUR-USERNAME.github.io/split-app**

> Замени `YOUR-USERNAME` на свой GitHub логин.

Установить как приложение на телефон:
- **Android:** Chrome → ⋮ → «Добавить на главный экран»
- **iPhone:** Safari → ⎙ → «На экран «Домой»»

---

### Локально (резервный вариант)

Скачай архив `СплитЧек_для_друзей.zip` из [Releases](../../releases).

Распакуй и запусти:
- **Windows:** двойной клик на `Открыть на Windows.bat`
- **Mac / Linux:** двойной клик на `Открыть на Mac.command`

> На Mac при первом запуске: правая кнопка мыши → «Открыть» → «Открыть»

## Деплой на GitHub Pages (пошагово)

### Шаг 1 — Создай репозиторий

1. Зайди на [github.com](https://github.com) → New repository
2. Название: `split-app`
3. Видимость: Public *(нужно для бесплатного Pages)*
4. Создать без README

### Шаг 2 — Настрой Pages

1. Settings → Pages
2. Source: **GitHub Actions**
3. Сохранить

### Шаг 3 — Загрузи код

```bash
# В папке проекта:
git init
git add .
git commit -m "initial"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/split-app.git
git push -u origin main
```

### Шаг 4 — Готово

Через ~2 минуты приложение будет доступно по адресу:
```
https://YOUR-USERNAME.github.io/split-app
```

Следующие обновления деплоятся автоматически при `git push`.

> ⚠️ Если имя репозитория отличается от `split-app`, измени `base` в `vite.config.ts`:
> ```ts
> base: '/ИМЯ-РЕПОЗИТОРИЯ/',
> ```

## Разработка

```bash
npm install      # установить зависимости (один раз)
npm run dev      # запуск с hot reload → http://localhost:5173
npm run build    # PWA сборка для GitHub Pages → dist/
npm run build:single  # один HTML файл → dist-single/
```

### Стек

| Слой | Технология |
|------|-----------|
| UI | React 19 + TypeScript |
| Сборщик | Vite 8 |
| Стили | Tailwind CSS 3 |
| База данных | Dexie.js (IndexedDB) |
| OCR | Tesseract.js |
| PWA | vite-plugin-pwa |
| Экспорт Excel | SheetJS |
| Экспорт JPG | html2canvas |

## Данные и приватность

Все данные хранятся **только на устройстве** в IndexedDB браузера.
Никаких серверов, никакой аналитики, никакой передачи данных.
